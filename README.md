# Renewable Energy Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing renewable energy data using **Power BI** and **Excel** to uncover insights into energy production, consumption, and trends.

The goal is to support data-driven decision-making in the renewable energy sector by visualizing key metrics such as generation patterns, sources, and efficiency.

---

## 📂 Files in Repository
- **AF05054491_06-03-2026.pbix** → Power BI dashboard file  
- **Renewable energy dataset (1).xlsx** → Dataset used for analysis  

---

## 📊 Key Features
- Interactive Power BI dashboard  
- Data cleaning and preprocessing using Excel  
- Visual insights on:
  - Energy production trends  
  - Renewable sources (solar, wind, hydro, etc.)  
  - Regional or time-based analysis  
- KPI tracking for performance evaluation
